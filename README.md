## Описание программы
** Training Planner ** - это GUI-приложение для планирования тренировок с фильтрацией, JSON и  Git.

### Возможности:
- ✅ Добавление тренировок
- ✅ Фильтрация тренировок
- ✅  Сохранение в JSON и загрузка данных из  JSON


## Инструкция по запуску
### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter 
-  Сохранение в JSON

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
3. Открыть файл  training_planner.py
4.Запустить приложение python training_planner.py
   

## Примеры использования

###Добавление тренировки
1.	Заполните поля:

-	Тип тренировки ("бег»)

-	Длительность («45»)

-	Нажать на кнопку «Добавить тренировку»

Тренировка появиться в таблицу

##Фильтрация

- Тип тренировок  

- Дата  
 Нажимаем кнопку «Применить фильтр»
Кнопка «сбросить» сбрасывает настройки фильтра
